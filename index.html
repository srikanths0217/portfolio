Projects

Network Credit System

A console-based Java application that creates a community-driven network performance rating system. Users can register and log in to report network issues, which dynamically affects the credit score of a specific provider. This system provides a clear, data-driven way for users to find the best network provider in a given area and for providers to identify and resolve problem zones.

Tech Used: Java

Key Features:
  • User Authentication: Implemented a secure registration and login system using mobile numbers.
  • Complaint Reporting: Users can log a complaint when a network is performing poorly.
  • Dynamic Credit System: Each network provider (e.g., Jio, Airtel, Vi) has a credit score that is automatically reduced based on user complaints.
  • Performance Insights: Allows users to check and compare network credit scores to easily determine which provider offers the best service in their location.
  • Data for Providers: Provides valuable, real-time feedback to network companies, helping them pinpoint areas needing network infrastructure improvements.
  • Object-Oriented Design: The system is built using classes to represent users, network providers, and complaints, demonstrating a strong grasp of OOP principles.
